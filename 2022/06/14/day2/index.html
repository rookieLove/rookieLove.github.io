MogDB-server 服务器

安装步骤

  1. 环境变量设置
    在/home/Dai/.bashrc中添加如下命令
  2. 编译配置
  3. 编译安装
    make -j8
    make install
    注意:清理编译
    make distclean

基本使用

  1. 初始化
    gs_initdb -D /home/Dai/data --nodename=mogdb

  2. 启动和终止
    gs_ctl start -D /home/Dai/data
    gs_ctl stop -D /home/Dao/date

  3. 查看相关进程
    ps -ef | grep Dai | grep -v grep | grep mogdb

  4. 日志存放位置
    /home/Dai/data/log_file

  5. 连接postgres
    gsql -d postgres -r

  6. 运行测试
    在MogDB-server/src/test/regress下运行
    make fastcheck_single
    单个测试：
    ./pg_regress -p 4096 hw_partition_hash_dml
